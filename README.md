# PWA2020
Programowanie aplikacji webowych 2020

## Volumes
* /DB/
* /code/

## Soft:

* Db: Sqlite3
* Web: Django
* Python: python3
* CMS: bootstrap4

## Prequsites:
* [GIT](https://git-scm.com/) <- git
* [Docker](https://www.docker.com/) <- docker with docker-compose module

## Run:
Git clone:
```
git clone https://github.com/Marynarz/PWA2020.git
```

cd:
```
cd PWA2020/
```

Docker-compose up
```
docker-compose up --build
```
